新用户注册流程 (意图先行重构版)
[cite_start]**调整背景**：为解决 5%-10% 用户在注册后未完善信息即流失的问题，将“注册-问卷-探索”流程重构为“意图先行-注册后置” [cite: 70, 71]。

### 全局主干流程
[cite_start]打开应用 -> 开屏动画 -> 选择登录/注册 -> 注册 -> AI 形象打招呼问候 -> **明确当下意图** [cite: 74, 75, 77, 78, 79, 82]。

根据用户意图，分流为以下三条支线：

#### [cite_start]支线 A：查找课程 (有基础健身目标) [cite: 81]
1. [cite_start]了解健身目标与运动类型偏好 [cite: 72, 73]。
2. [cite_start]输出计划预期 -> 选择教练 -> 生成今日训练 -> 开始运动 [cite: 83, 86, 103, 105]。
3. [cite_start]**注册，完善基础信息** -> 进入首页生成首日任务 [cite: 107, 110, 111]。

#### [cite_start]支线 B：连接设备 (有强目标) [cite: 88]
1. [cite_start]了解设备类型 -> 获取相关权限 -> 扫描并连接设备 [cite: 89, 91, 92]。
2. [cite_start]**若连接成功**：基础使用介绍 -> 已掌握 -> 开始第一次锻炼 -> **注册，填写基础信息** -> 选择虚拟教练 -> Free Mode 锻炼 -> 创建计划 -> 进入首页 [cite: 98, 99, 100, 112, 115, 116, 118, 120, 114]。
3. [cite_start]**若连接失败**：权限/设备状态确认 [cite: 84, 85]。
   * [cite_start]若未解决，进行型号鉴别 [cite: 87, 96]。
   * [cite_start]鉴别为可连接：转入人工/AI 客服支持 [cite: 104, 108]。
   * [cite_start]鉴别为不可连接：提供课程内容/踏频器指引 [cite: 106, 109]。

#### [cite_start]支线 C：随便看看 (无目标) [cite: 90]
1. [cite_start]通用的场景/挑战推荐 -> 选用 [cite: 93, 94]。
2. [cite_start]完善偏好信息 -> 选择虚拟教练 [cite: 95, 101]。
3. [cite_start]**注册，完善基础信息** -> 进入首页生成首日任务 [cite: 102, 110, 111]。
